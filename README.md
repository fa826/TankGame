# Required Information when Submitting Tank Game

## Version of Java Used:
    openJDK : 24

## IDE used: 
    IntelliJ IDEA 2025.1.1.1 (Ultimate Edition)

## Steps to Import project into IDE:
    1. Open IntelliJ IDEA.
    2. Open the folder containing the project.
    3. Make sure the Project SDK is set to openJDK: 24.
    4. Mark the resources directory to the "Test Resources Root".

## Steps to Build Your Project:
    1. Open the Build menu from the top.
    2. Select Build Project.
    3. Wait for IntelliJ to compile the code, it will show BUILD SUCCESSFUL.

NOTE: Make sure all images/sound resources are in the "resources" folder and are loaded correctly.
 
## Steps to run your Project:
    1. Locate Launcher.java (/src/tankgame/game/Launcher.java).
    2. Right-click on the file and choose Run 'Launcher.main()'.
    3. The game window will appear and launch the Start Menu.
    4. Select the map of your choice and click Start to begin playing.
